# IranMap

Real-time Iran conflict monitoring platform powered by OSINT and geospatial intelligence.

## Features
- Live map visualization
- Layer-based filtering
- Conflict zone tracking
- Military & infrastructure data

## Tech Stack
- Vercel (hosting)
- Cloudflare (DNS/CDN)
- Leaflet (map)
- OpenStreetMap

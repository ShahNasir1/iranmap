# IranMap

Real-time geospatial intelligence platform for monitoring, visualization, and analysis of conflict zones using OSINT data.

🌐 Live: https://iranmap.online

---

## 🧠 Overview

IranMap is an interactive mapping platform designed to track and analyze conflict-related activities through geospatial visualization.

The platform combines open-source intelligence (OSINT), mapping technologies, and real-time data structuring to provide a clear operational picture.

---

## 🚀 Features

- 🗺️ Interactive map visualization
- 🎯 Layer-based filtering system
- 🔍 Search & location lookup
- 📊 Conflict zone tracking
- 🛰️ Military & infrastructure mapping
- 🌙 Dark / Light theme support
- 📥 KMZ file import support
- 📸 Map export (image capture)

---

## 🛠️ Tech Stack

**Frontend**
